##  Project: MediaTech City Platform - City Guide & Events (TG-бот)

>**MediaTech / продукт "места и события" - Web + Telegram-бот на общей модели данных**

**Суть:** подготовить консистентный пакет аналитических артефактов по ветке Telegram-бот, достаточный для реализации MVP без повторного пересбора требований и без разрозненных трактовок. В пакете увязаны требования, границы ответственности Telegram - backend - БД, потоки данных, поведение сценариев и интеграционные контракты.

**Результат:** набор артефактов "данные - процессы - поведение - интеграция" + слой трассировки решений (Q&A и метки), чтобы отделять зафиксированные правила от зависимостей/расширений.

>**Инструменты и нотации:** dbdiagram (DBML/ERD), draw.io (DFD), UML Sequence (PlantUML), Markdown/PDF.

>**План размещения:** в репозитории хранятся PDF-версии и исходники (MD/PUML), для диаграмм - ссылки на онлайн-просмотр.
